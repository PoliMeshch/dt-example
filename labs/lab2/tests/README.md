# Тестовые файлы лабораторной работы №2

## Структура
- `temp/` - временные файлы (автоматически очищаются)
- `fixtures/` - постоянные тестовые данные

## Файлы по заданиям
- `2_4_*` - задание 2.4
- `2_5_*` - задание 2.5  
- `3_2_*` - задание 3.2
- `3_3_*` - задание 3.3

Все файлы автоматически удаляются командой `make clean`
