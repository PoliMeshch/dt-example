### Этап 2. Изучение особенностей дистрибутива (KDE Neon)

**Разработчик и поддержка:** Сообщество KDE при поддержке компании Blue Systems.

**Базовый дистрибутив:** Ubuntu LTS (Long Term Support).

**Релизы и последняя версия:** KDE Neon — это rolling release для окружения KDE Plasma (обновления выходят 
постоянно). Последняя стабильная версия — на основе Ubuntu 24.04 LTS.

**Ресурсы для изучения:** 
- Официальный сайт: [https://neon.kde.org/](https://neon.kde.org/)
- Документация KDE: [https://community.kde.org/](https://community.kde.org/)
- Форумы: [https://discuss.kde.org/](https://discuss.kde.org/)

**Пакетный менеджер:** APT (Advanced Package Tool).

**Основные команды:**
- Поиск пакета: `apt search <имя>`
- Установка: `sudo apt install <имя>`
- Обновление списка пакетов: `sudo apt update`
- Обновление системы: `sudo apt upgrade`
- Удаление: `sudo apt remove <имя>`

**Процесс установки:** 

Загрузка ISO-образа, создание загрузочной флешки (или подключение в VirtualBox), 
запуск установщика, выбор языка, разметка диска, создание пользователя. 
Официальная инструкция: [https://neon.kde.org/](https://neon.kde.org/)

**Минимальные системные требования (KDE Neon + KDE Plasma):**
- Процессор: 1 ГГц (рекомендуется 2 ГГц)
- ОЗУ: 2 ГБ (рекомендуется 4 ГБ)
- Диск: 20 ГБ (рекомендуется 25 ГБ)

### 4. Настройка и установка пакетов

#### 4.1. Управление окнами и переключение между программами
В KDE Plasma можно открывать несколько приложений через меню «Пуск». Переключение между 
окнами — через панель задач или комбинацией `Alt + Tab`. На скриншоте показаны одновременно 
открытые терминал, файловый менеджер и браузер.

![alt text](imgs/lab4_imgs/windows-switch.png)

#### 4.2. Отслеживание системных ресурсов и управление процессами
Установлен пакет `htop`:

![alt text](imgs/lab4_imgs/htop-install.png)

Запуск `htop` в терминале показывает загрузку CPU, памяти и список процессов:

![alt text](imgs/lab4_imgs/htop.png)

- Верхняя часть показывает загрузку процессора (CPU), оперативной памяти 
(Mem) и своп-раздела (Swap). Полосы разного цвета — разные типы нагрузки (например, 
системные процессы, пользовательские, ожидание ввода-вывода).
- Нижняя панель - это меню быстрых действий (F1...F10).
- Таблица процессов: cписок всех запущенных программ, их ID (PID), загрузка ЦП (%CPU), 
потребление памяти (%MEM) и команда.

Завершение процесса — клавиша `F9`. Нужно навести курсор на процесс и в открывшемся списке 
выбрать `SIGTERM`, затем нажать `Enter`:

![alt text](imgs/lab4_imgs/sigterm.png)

Команды `free -h` и `df -h` показывают память и место на диске:
